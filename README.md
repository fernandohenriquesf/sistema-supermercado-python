
# Sistema de Supermercado em Python

## Descrição

Sistema de supermercado desenvolvido em Python utilizando CustomTkinter e SQLite.

Funcionalidades implementadas:

* Login de usuário
* Busca por nome do produto
* Busca por código de barras
* Carrinho de compras
* Controle de estoque
* Registro de vendas
* Backup automático
* Cadastro inicial de produtos

## Tecnologias

* Python 3.12
* SQLite
* CustomTkinter
